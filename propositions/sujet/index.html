Proposer un sujet

Nous proposons 2 formats différents :

  • Le court (5 minutes), par exemple pour susciter la curiosité en faisant découvrir une librairie, ou pour parler d’un sujet précis lié à la programmation.
  • Le long (20 minutess + 5 minutes de questions), parfait pour rentrer un peu plus dans le sujet, développer l’argumentation, faire un “live-coding”.

Retrouvez tous les talks et votez pour votre préféré sur Github
